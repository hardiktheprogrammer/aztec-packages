## scalar_mul::multi_scalar_mul_embedded_curve

Performs multi scalar multiplication over the embedded curve whose coordinates are defined by the configured noir field.
For the BN254 scalar field, this is BabyJubJub or Grumpkin.
Suffixes `_low` and `_high` denote low and high limbs of the input scalar.
The function accepts arbitrary amount of point-scalar pairs on the input, it multiplies the individual pairs over
the embedded curve and returns a sum of the resulting points.

Points represented as x and y coordinates [x1, y1, x2, y2, ...], scalars as low and high limbs [low1, high1, low2, high2, ...].

#include_code multi_scalar_mul_embedded_curve noir_stdlib/src/scalar_mul.nr rust

example

```rust
fn main(point_x: Field, point_y: Field, scalar_low: Field, scalar_high: Field) {
    let point = std::scalar_mul::multi_scalar_mul_embedded_curve([point_x, point_y], [scalar_low, scalar_high]);
    println(point);
}
```
